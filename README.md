# PhD Thesis Application

This repository contains the microbiome data analysis conducted as part of a PhD thesis. It analyses gut microbiota data from the PASTURE birth cohort study, focusing on samples collected at 2 months of age from infants in Germany, Switzerland, and Austria. The analyses examine alpha diversity and its associations with clinical and environmental variables such as birth mode, breastfeeding, maternal smoking during pregnancy, and number of siblings.

The workflow is organized using the [projthis](https://github.com/ijlyttle/projthis) R package, which provides a lightweight framework for managing data flow between analysis steps. Each R Markdown file writes its outputs to a numbered target directory under `workflow/data/`, which the next file can read as a source.

## Repository Structure

```plaintext
.
├── projthis_code.R               → Helper script for rendering workflow Rmd files
├── README.md                     → Project overview (this file)
├── renv.lock                     → Lock file for renv environment
│
├── workflow/                     → Analysis workflow (R Markdown files)
│   ├── 00_build_dataset          → Load raw PASTURE data, filter samples/variables,
│   │                               and export clean phyloseq objects
│   │
│   ├── 00_phyloseq_objects       → Inspect and document phyloseq objects
│   │
│   ├── 01_overview               → Descriptive overview of the study cohort and
│   │                               microbiome composition
│   │
│   ├── 02_alpha_diversity        → Alpha diversity analysis (richness, Shannon,
│   │                               Gini-Simpson) across clinical/environmental features
│   │
│   ├── data/                     → Inputs and outputs of the Rmd files (managed by projthis)
│   │   ├── PASTURE_data/         → Raw input data (not published)
│   │   ├── 00_build_dataset/     → Outputs of 00_build_dataset.Rmd
│   │   ├── 01_overview/          → Outputs of 01_overview.Rmd
│   │   └── 02_alpha_diversity/   → Outputs of 02_alpha_diversity.Rmd
│   │
│   └── figures/                  → Figures generated by the Rmd files
│
└── renv/                         → R environment for reproducibility
```

## Reproducibility

Package dependencies are managed with [renv](https://rstudio.github.io/renv/). To restore the environment:

```r
renv::restore()
```

To render the full workflow in order, source `projthis_code.R`.
